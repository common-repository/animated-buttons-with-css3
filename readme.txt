== Description ==

Animated Buttons with CSS3 for you to use in your post, you can use them to download buttons, buttons for hyperlinks (links), or for whatever you want =)

 

= Features: =

 - Colors: Blue, Orange, Grey and Green
 - Sizes: Small, Medium and Large
 - Auto width
 - Shordcode
 - Button on your panel
 - Animation 

Botones animados con CSS3 para que los uses en tus post, puedes usarlos para botones de descargas, botones para hipervinculos (links), o para lo que quieras =) 

 

= Caracteristicas: =

 - Colores: Azul,Naranja, Gris y Verde
 - Tamaños: Pequeños, Medianos y Grandes
 - Ancho auto
 - Shordcode
 - Boton en tu panel
 - Animacion 


[DEMO](http://nexxuz.com/animated-buttons-with-css3-plugin-wordpress-para-generar-botones-animados.html)


== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
